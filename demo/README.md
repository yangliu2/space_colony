# O'Neill Cylinder Constraint Explorer

Interactive demo for exploring habitat design constraints with real-time 3D visualization.

## Running

```bash
# Terminal 1 — API server (port 8042)
conda activate space
cd models/habitat_constraints
uv run uvicorn habitat_constraints.api.main:app --host 127.0.0.1 --port 8042 --reload

# Terminal 2 — Frontend dev server (port 5173)
cd demo
npm install   # first time only
npm run dev
```

Open http://localhost:5173 in your browser.

## Limitations

- **Brave Browser**: The Three.js 3D cylinder scene does not render correctly in Brave. Brave's Shields (fingerprinting protection) can block or interfere with WebGL rendering. Use **Safari** or **Chrome** instead.
